# Pilot Launch Plan

Validate SmartShop **FREE** before Premium AI.

## Pilot city

**St. Pölten**, Austria

## Target users

- Arabic families
- Turkish families

These communities frequently rely on manual restaurant street-board offers, word of mouth, and local shop flyers — ideal for validating the free-tier manual offers architecture.

## Pilot goal

Prove that families can:

1. Register and maintain a family profile
2. Build a manual local offers database (supermarkets + restaurants)
3. Generate compact rule-based shopping baskets
4. Return weekly without Premium AI

Premium AI remains **out of scope** for the pilot.

## Metrics

| Metric | Description |
|--------|-------------|
| Registered families | Total family accounts created |
| Active users | Families with activity in the last 7 days |
| Manual supermarket offers | Retail offers added via manual entry or photo import |
| Manual restaurant offers | Restaurant street-board / phone / word-of-mouth offers |
| Shopping baskets created | Compact baskets generated by deterministic rules |
| Favourite stores | Store profiles marked as favourites |
| Favourite restaurants | Restaurant profiles marked as favourites |
| Weekly retention | % of families active week-over-week |
| User feedback | Community feedback reports (future architecture) |

## Architecture in scope

- `StoreProfile` and `RestaurantProfile` merchant registries
- Manual offers referencing profiles (no duplicated merchant data)
- Offer confidence and status tracking
- Compact shopping basket contracts
- Deterministic basket rules only

## Architecture out of scope

- OpenAI / LLM / AI inference
- Payment and subscriptions
- Authentication backend
- OCR runtime
- Maps and coordinates runtime
- External APIs and crawling
- Premium AI features

## Success criteria

- Families in St. Pölten can manually maintain offers for local Arabic and Turkish restaurants
- Baskets stay compact and useful without merchant detail clutter
- Weekly retention demonstrates habit formation before Premium AI investment
