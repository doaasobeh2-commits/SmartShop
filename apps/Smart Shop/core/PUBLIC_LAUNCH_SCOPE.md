# Public Launch Scope (v1)

SmartShop AI launches first as a **shopping app only**.

## Visible in v1

- Smart shopping
- Local offers
- AI shopping basket
- Family preferences
- Saved shopping history

## Hidden until later

- Recipe AI screens and features
- Fitness AI screens and features
- Inventory UI
- Meal selection UI
- Weekly inventory report UI

Inventory intelligence is architected internally but must not appear in the product UI yet.

See [HIDDEN_INVENTORY_LEARNING.md](./HIDDEN_INVENTORY_LEARNING.md).
